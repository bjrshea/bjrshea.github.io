VetValuation

Built for owners of veterinarian clinics, VetValuation is a program that calculates the financial valuation for an owner's veterinary practice.

After the user decides how granular the valuation should be, the calculators in this program compare a host of variables (sometimes up to 30), factor in the practice's EBITDA, and use conditional logic to choose a multiplier that will give the practice the most accurate valuation. The calculators run almost primarily on JavaScript and some PHP.

This project is currently pending approval from the client's board of directors, but I'd be happy to provide you with a demo if you'd like. Just head down to the bottom of this page and drop me a line!

iVET360's
Vet Marketing Report

Prior to the Vet Marketing website, reporting for the iVET360 marketing team was done via PDFs. Not only was this a tedious process for the team, but it also meant clients had to scroll through loads of text everytime they got a report.

To build this report, I used a heavy dose of PHP, JavaScript, and CSS. I passed the data from the forms to the results pages via a query string, and I grabbed the variables from the query string using PHP GET requests. From there, I used JavaScript to compare the data points to their respective benchmark and some CSS to give the data a little style.

Feel free to generate a fake report and see how it works! I recommend either the Benchmark or Scorecard Report to get a feel for how much information used to go into the old PDFs.
